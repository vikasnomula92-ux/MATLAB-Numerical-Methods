# MATLAB Numerical Methods Implementation
## Description
This project implements fundamental numerical methods using MATLAB to solve mathematical and engineering problems. The algorithms are developed from scratch without relying on built-in numerical solvers, providing a clear understanding of numerical analysis concepts.
## Features
- Newton-Raphson Method
- Bisection Method
- Secant Method
- Trapezoidal Rule
- Simpson's Rule
- Numerical Differentiation
- Root Finding
- Function Plotting
- Error Analysis
## Technologies Used
- MATLAB

## How to Run
1. Open MATLAB.
2. Open the project folder.
3. Navigate to the **Codes** folder.
4. Run the required MATLAB file. For example:
```matlab
newton_raphson
```
or
```matlab
trapezoidal_rule
```
